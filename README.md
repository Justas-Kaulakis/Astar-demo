# Astar-demo

This is an interactive A* algorithm visualization on a map.  
Made with Vite, Typescript, leafletjs.  

Inspired by a [Youtube video](https://www.youtube.com/watch?v=CgW0HPHqFE8) by 'ones and zeros'.
